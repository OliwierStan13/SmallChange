Project Overview

Devs: Stephen Mac Partlin & Oliwier Stanislawczyk   Advisor: Derry Bradley   PO: Louise O'Donovan


For this Small Change project we explored the main web development languages of HTML, JavaScript and CSS. We developed tasks for use case items UC-04 (Portfolio page) and UC-05 (Trading page). We completed this small website alongside the training material on Pluralsight, which gave us a better understanding of these core web development practices. We will use this experience as a learning opportunity and how to effectively work as a team.


Also as part of this project we worked as a team and used a code collaboration tool called GitHub. GitHub gave us a great opportunity to use a widely popular version control tool and that was how we kept the code maintained between the team. Also as part of this we learned key values of teamwork and learned quite a bit from one another in terms of good practices and good habits when it comes to web design. As a team, we worked efficiently and tried to help each other out where possible.
